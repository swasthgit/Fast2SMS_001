## Step 3: Send SMS

1. **Download Sample CSV** from the home page
2. **Fill your data** in the CSV file:
   - Mobile numbers (10 digits, no +91)
   - Variables (v1, v2, v3, etc.)
   - Template ID (from the list on home page)
3. **Upload the CSV** file
4. **Enter your Fast2SMS API key**
5. **Click "Send"**
6. **Download results** as Excel log file
